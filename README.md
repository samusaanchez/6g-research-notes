## Scope

Analysis of the use-case families defined under the IMT-2030 framework and the requirements that separate them from 5G capabilities, including:

- **Holographic telepresence & Internet of Senses** — multi-view/light-field transmission at 1 Gbps–1 Tbps; <1 ms end-to-end latency for haptic feedback; 1–10 kHz synchronization update rates
- **Massive city-scale Digital Twins** — over-the-air delays below 0.1 ms, sub-100 ns synchronization accuracy, centimeter-level positioning, tight integration of sensing and communication (ISAC)
- Additional IMT-2030 scenario families covered in the full document

The emphasis is on **why each requirement exists** (sensory dissonance thresholds, twin-synchronization physics, positioning accuracy for mobile asset mapping) rather than on listing headline figures.
